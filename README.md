# VIT-event-registration-portal
A Event registration portal for VIT made in React, Express and postgresSQL


## To run the code

- Create a database  and add details in db.js
```sql

host: "",
port: "",
user: "",
password: "",
database: "",

```
- Add node modules in express
```
npm init
```

- Start server
```
npm run dev
```

- Start react frontend
```
yarn install
yarn start
```
